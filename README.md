# By Product DashBoard — Project templates

This repository contains a Vite + React starter and a set of workspace support files, editor recommendations, and small reusable templates to speed design & development.

What's included
- Editor and workspace configs: `.editorconfig`, `.gitignore`, `.prettierrc`, `.eslintrc.json`
- VS Code recommendations: `.vscode/extensions.json`, `.vscode/settings.json`
- Small templates: component, module CSS, hook, page
- Test config template: `vitest.config.js` and `src/setupTests.js`

Next steps
1. Install recommended dev dependencies:

```powershell
npm install --save-dev eslint prettier vitest @testing-library/react @testing-library/jest-dom @testing-library/user-event
```

2. (Optional) Install recommended VS Code extensions from `.vscode/extensions.json`.
